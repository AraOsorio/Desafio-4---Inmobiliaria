# 🏠 Desafío Inmobiliaria

## 📌 Descripción
Proyecto web desarrollado con HTML, Bootstrap y JavaScript para mostrar propiedades en venta y alquiler.

Se utilizaron arreglos, objetos, ciclos `for` y condicionales para renderizar tarjetas dinámicamente y mostrar si permite fumar o mascotas.

🔗 Demo del proyecto:  
https://araosorio.github.io/Desafio-4---Inmobiliaria/

---

## 🚀 Funcionalidades

- Página principal con tarjetas de propiedades.
- Página de propiedades en venta.
- Página de propiedades en alquiler.
- Render dinámico con JavaScript.
- Indicadores de:
  - 🚭 Permite fumar
  - 🐾 Permite mascotas
